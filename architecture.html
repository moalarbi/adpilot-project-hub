Architecture

التدفق الأساسي

User Input Layer → AI Layer → Ads Engine → Optimization Engine → Analytics & Leads Layer

User Layer واجهة بسيطة جدًا عبر الويب أو الجوال لإدخال النشاط، المدينة، الميزانية، والهدف.
AI Layer LLM لتحليل المدخلات، بناء الاستراتيجية، اقتراح الجمهور، وتوليد النصوص الإعلانية.
Ads Engine ربط API مباشر لإنشاء الحملات، جدولة الإطلاق، وسحب نتائج الأداء.
Analytics / CRM تخزين النتائج، تتبع الـ Leads، وإرسال ملخصات الأداء اليومية أو الأسبوعية.

Tech Stack المقترح

  • Frontend: Next.js
  • Mobile لاحقًا: React Native
  • Backend: Node.js + TypeScript
  • Database: PostgreSQL
  • Cache / Queue: Redis
  • AI: OpenAI / Anthropic
  • Infra: AWS أو GCP
